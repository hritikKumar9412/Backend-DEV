# Linux Commands and Git Commands

## Linux Commands
-----------------------------------------------------------------------------
| Command                      | Description                                 |
|------------------------------|---------------------------------------------|
| `ls`                         | List directory contents                     |
| `cd`                         | Change directory                            |
| `pwd`                        | Print working directory                     |
| `mkdir <dir>`                | Create a new directory                      |
| `rm <file>`                  | Remove a file                               |
| `rm -r <dir>`                | Remove a directory and its contents         |
| `cp <src> <dest>`            | Copy files or directories                   |
| `mv <src> <dest>`            | Move/rename files or directories            |
| `cat <file>`                 | Display file contents                       |
| `touch <file>`               | Create an empty file                        |
| `echo <text>`                | Display a line of text                      |
| `man <command>`              | Show manual for a command                   |
| `chmod <mode> <file>`        | Change file permissions                     |
| `chown <owner> <file>`       | Change file owner                           |
| `grep <pattern> <file>`      | Search for a pattern in a file              |
| `find <dir> -name <pattern>` | Find files by name                          |
| `top`                        | Display running processes                   |
| `ps`                         | Show process status                         |
| `kill <pid>`                 | Kill a process by PID                       |
| `df -h`                      | Show disk space usage                       |
| `du -sh <dir>`               | Show directory size                         |
| `ifconfig` or `ip a`         | Show network interfaces                     |
| `wget <url>`                 | Download files from the internet            |
| `tar -xzvf <file.tar.gz>`    | Extract a tar.gz archive                    |
-----------------------------------------------------------------------------


## Git Commands

-------------------------------------------------------------------------------
| Command                        | Description                                 |
|--------------------------------|---------------------------------------------|
| `git init`                     | Initialize a new git repository             |
| `git clone <repo>`             | Clone a repository                          |
| `git status`                   | Show status of working directory            |
| `git add <file>`               | Add file(s) to staging area                 |
| `git commit -m "msg"`          | Commit staged changes with a message        |
| `git push`                     | Push commits to remote repository           |
| `git pull`                     | Fetch and merge changes from remote         |
| `git branch`                   | List branches                               |
| `git branch <name>`            | Create a new branch                         |
| `git checkout <branch>`        | Switch to a branch                          |
| `git merge <branch>`           | Merge a branch into current branch          |
| `git log`                      | Show commit history                         |
| `git remote -v`                | Show remote URLs                            |
| `git fetch`                    | Download objects and refs from remote       |
| `git reset --hard <commit>`    | Reset to a specific commit                  |
| `git rm <file>`                | Remove a file from the repository           |
| `git stash`                    | Stash changes in a dirty working directory  |
| `git tag <name>`               | Create a tag                                |
| `git show <commit>`            | Show details of a commit                    |
-------------------------------------------------------------------------------
